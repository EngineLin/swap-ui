# SWAP-UI

SWAP UI Framework. www.swap.work

基於 [@material-ui/core](https://material-ui.com/) 的前端框架來開發專屬 SWAP 的介面元件，達到介面上的一致性與開發上的效率。

# 安裝

```
$ yarn add react react-dom styled-components
```

```
$ yarn add @yosgo/swap-ui
```

# 開發

```
$ yarn storybook
```

```
$ npm publish
```
